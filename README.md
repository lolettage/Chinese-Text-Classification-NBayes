
--------------------------------------
对原始训练数据的处理: 
原始的txt文件都被按行分割为了多个小txt文件保存在相应的类别目录下.

--------------------------------------
训练过程共包括四步:
(1) 用jieba分词.
(2) 把分好的词转换为sklearn库中的Bunch格式.
(3) 用TF-IDF构建有权词向量空间.
(4) 用朴素贝叶斯算法对test数据进行分类.
